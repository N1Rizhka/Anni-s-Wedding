# Ani and Zviadi Wedding Invitation

Single-page wedding invitation website with:

- animated envelope opening interaction
- wedding details for 5 September at 17:00
- venue section with map link
- summer chic dress code section
- client-side RSVP form that saves responses in the browser

## Preview locally

Open `index.html` directly in a browser, or serve the repository with a simple static file server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
