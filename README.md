# Ani & Zviadi — Wedding Invitation

An elegant online wedding invitation with an interactive envelope opening experience and RSVP form.

## Features

- **Interactive envelope** — Tap to open a wax-sealed envelope revealing the invitation
- **Wedding details** — Date, time, venue, and Summer Chic dress code
- **RSVP form** — Guest name, attendance, guest count, dietary requirements, and message
- **Summer Chic design** — Bright coral, gold, sage, sky, and lavender palette with elegant typography

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

Static files are output to `dist/`.

## RSVP Data

RSVP responses are stored in the browser's `localStorage` under the key `wedding-rsvps`. For production, connect the form to a backend service (e.g. Formspree, Google Sheets, or a custom API).
