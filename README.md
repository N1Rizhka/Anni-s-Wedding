# Ani & Zviadi Wedding Invitation

A bright, summer-chic online wedding invitation with an animated envelope reveal,
brochure-style event details, and a client-side RSVP form.

## Wedding details

- Date: 5 September
- Time: 17:00
- Venue: Garden of Ilia
- Address: Davit Aghmashenebeli Avenue, 73, Tbilisi
- Dress code: Summer Chic - bright colors, elegant, joyful summer spirit

```bash
npm install
npm run dev
```
